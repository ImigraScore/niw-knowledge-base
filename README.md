# EB-2 NIW Knowledge Base

> An open-access analytical hub for interpreting U.S. EB-2 National Interest Waiver adjudication logic. Maintained by ImigraScore.

---

## Strategic Purpose

The **NIW Knowledge Base** serves as the interpretative layer of the [EB-2 NIW Technical Framework](https://github.com/ImigraScore/eb2-niw-framework). While the Framework provides the technical standards and structural methodology, this Knowledge Base provides structured analysis of specific adjudication patterns, policy updates, and strategic considerations.

Our goal is to translate abstract legal standards into reproducible analytical models.
